# electron-react-typescript-template

Minimal template for app build on Electron, React and TypeScript.

# Features

    - Mocha integration for testing.
    - Live reload for dev and testing.
    - Code coverage.

# Install deps

    npm install

# Compile TypeScript

    npx tsc

# Run tests

    npm run test
    npm run test:watch

# Run for development

    npm start


# Build the installer

    npm run build
